Welcome to our RESTful API service

Available Endpoints

API Root: GET /api/
Main API information
User Registration: POST /api/users/
Create a new user account
Login: POST /auth/token/login/
Get authentication token
User Profile: GET /api/users/me/
Get current user profile (requires token)
Categories: GET /api/categories/
List all food categories
Menu Items: GET /api/menu-items/
List all menu items
Ratings: GET/POST /api/ratings/
View or create ratings
